## 重要な事実・固有名詞（確認済み）

### 地質

| 項目 | 正しい記述 |
|---|---|
| 地層名 | 熱田層（洪積層 / 更新世）|
| 二層構造 | 上部熱田層（砂・礫主体）・下部熱田層（粘土・シルト主体）|
| N値 | 低め（洪積層としては高くない）。ただし**同じN値でも沖積砂より液状化強度が高い** |
| PL値の判定基準 | 5超：液状化危険度 **高い** / 15超：液状化危険度 **極めて高い** ※「15以上で高リスク」は誤り |
| 液状化リスクが低い理由 | 年代効果（aging effect）：長期の圧密・セメンテーション・応力履歴により洪積砂は液状化抵抗が高い |
| 台地の東縁 | 新堀川（旧・精進川） |
| 台地の西縁 | 堀川（名古屋城築城（1610年）時に開削した人工運河） |

### 歴史・地理

| 項目 | 正しい記述 |
|---|---|
| 名古屋城 | 1610年築城開始・1612年天守完成 |
| 堀川の機能 | 防御だけでなく、熱田湊から名古屋城への物資輸送路として機能 |
| 名古屋駅開業 | 1886年（明治19年） |
| 名古屋駅の立地経緯 | 当時の市街地（熱田台地上）から外れた低湿地。用地取得のしやすさ・市街地との調整コスト削減が理由 |
| 地盤沈下 | 大正〜昭和期の地下水汲み上げによる広範囲の地盤沈下が、沖積低地の液状化リスクをさらに高めた |

### データセンター（熱田台地上に立地）

| 施設名 | 運営会社 | 備考 |
|---|---|---|
| ctcデータセンター名古屋丸の内 / 名古屋栄 | **中部テレコミュニケーション株式会社** | ※「伊藤忠テクノソリューションズ」は誤り |
| NTT・KDDIなど通信系各社のDC群 | 各社 | 個別施設名は確認の上で追記 |
| メイテツコム熱田データセンター | メイテツコム | 施設名に「熱田」が入っている点が口頭説明のネタになる |

## 禁止事項・注意事項

- **PL値の閾値を「15以上で高リスク」と記述しない**（正：5超で高い・15超で極めて高い）
- **ctcを「伊藤忠テクノソリューションズ」と記述しない**（正：中部テレコミュニケーション）
- 熱田層を単一層として説明しない（上部・下部の二層構造がある）
- 液状化リスクが低い理由を「N値が高いから」と説明しない（N値は低め。年代効果が理由）
- 自己紹介スライドは含めない
- 発表者名・イベント名・日付はプレースホルダー `（名前）` `（イベント名）` `（日付）` のままにする
